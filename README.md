# Heart Disease Prediction Task 3

This project is a Machine Learning model that predicts whether a patient has heart disease or not.

## Steps:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Logistic Regression Model
- Confusion Matrix
- ROC Curve

## Output:
Binary classification (0 = No Disease, 1 = Disease)

## Model Used:
Logistic Regression
